- block contract 採 Top 強控型（base/cfg/token_range/tile_range/phase_id）。
- single-X_WORK baseline 已凍結；W_REGION 與 runtime scratch 不 alias。
- FinalHead 正式為 Pass A / Pass B，Pass bridge 落於 SCRATCH.FINAL_SCALAR_BUF。
- READ_MEM state gating 已凍結：IDLE/HALTED 合法；active state 回 RSP_ERR(ERR_BAD_STATE)，避免 data_out 混流。
- ternary SRAM storage 規格摘要已凍結：
  - codebook：00=0、01=+1、11=-1、10 illegal
  - 2-bit packing、32-bit word=16 weights、LSB-first
  - last_word_valid_count=1..16（滿載=16）、tail padding=0
  - per-matrix inv_s_w、matrix record（logical section format，非手動固定 physical base offset）
  - W[out][in] 為數學表示；in contiguous 僅指實體線性化
